# My portfolio

My portfolio built using React JS


## Demo

https://my-portfolio-abhinav2011.vercel.app/

## Run Locally

Clone the project

```bash
  git clone https://github.com/hitarth031099/my-portfolio
```

Go to the project directory

```bash
  cd my-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
